# Demo Workflow

Use this flow for the final project demonstration.

1. Run `PYTHONPATH=src python scripts/smoke_check.py`.
2. Open the app with `PYTHONPATH=src streamlit run app/streamlit_app.py`.
3. Show Home page: project goal, open-source model approach, architecture.
4. Open Settings: show configurable model and generation defaults.
5. Open Generate Learning Material.
6. Enter a topic, for example `Photosynthesis`.
7. Select difficulty, language, output length, quiz count, and generation parameters.
8. Generate content.
9. Show structured sections, flashcards, quiz, mini project, roadmap, and references.
10. Open History and show saved generations.
11. Open Downloads and export Markdown, TXT, HTML, and PDF.
12. Show data pipeline documentation.
13. Show evaluation report output under `outputs/evaluation/`.

If model generation is too slow during presentation, explain that the architecture supports local open-source inference and use a saved history item for UI/export demonstration.
